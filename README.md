## how to use

install pip:

Linux
`sudo apt-get install python3-pip`

windows
`get-pip-py`

## Install requirements

Linux
`pip install time, threading, pynput`

windows
`python -m pip install time, threading, pynput`

## run

Linux
`python3 dogeminer-bot.py`

windows
`python dogeminer-bot.py`
